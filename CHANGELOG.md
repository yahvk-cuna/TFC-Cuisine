# 0.2.0

## Features
*   Introduced configurable formulas for food hunger, nutrition, saturation, and water content.
*   Expanded support for various blocks as heat sources.
*   Added recipe to clean plates.
*   Integrated Firmalife chocolate as a usable ingredient in recipes.

## Bug Fixes
*   Fixed an issue where placing cooking implements would extinguish the Charcoal Forge.

## Improvements
*   Improved compatibility with older Forge versions.
*   Enhanced clarity of configuration comments for better user understanding.

## Miscellaneous
*   Added a mod description.
