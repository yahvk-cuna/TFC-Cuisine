# TFC Cuisine
Compatible mod for TerraFirmaCraft and Cuisine Delight.

## Feature (configurable)
* Support putting TFC food in the Cuisine Skillet.
* Calculate servings and nutritional values dynamically using a similar method to TFC soups.
* Raw/Overcooked ingredients will affect nutrition.
* Allow heating the Skillet using a TFC heat source (e.g. oven bottom in Firmalife).
* The spatula can be put on the tool rack.

## Support Mod
* TerraFirmaCraft
* Firmalife
* Beneath
